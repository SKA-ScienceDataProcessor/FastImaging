Detailed Description

The SourceFindImage class for source detection.

The structure collects intermediate results from source-detection. This can be useful for verifying / debugging the sourcefinder results, and intermediate results can also be reused to save recalculation.

SourceFindImage constructor.

Constructs SourceFindImage structure and detects positive and negative (if input_find_negative_sources = true) sources

Parameters
[in]input_data(arma::Mat): Image data.
[in]input_detection_n_sigma(double): Detection threshold as multiple of RMS
[in]input_analysis_n_sigma(double): Analysis threshold as multiple of RMS
[in]input_rms_est(double): RMS estimate (may be 0.0, in which case RMS is estimated from the image data).
[in]find_negative_sources(bool): Find also negative sources (with signal is -1)
[in]sigmaclip_iters(uint): Number of iterations of sigma clip function.
[in]median_method(MedianMethod): Method used to compute the median.
[in]gaussian_fitting(bool): Perform gaussian fitting for each island.
[in]generate_labelmap(bool): Update the final label map by removing the sources below the detection threshold.
[in]ceres_diffmethod(CeresDiffMethod): Differentiation method used by ceres library for gaussian fitting.
[in]ceres_solvertype(CeresSolverType): Solver type used by ceres library for gaussian fitting.
