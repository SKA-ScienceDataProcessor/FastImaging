Detailed Description

Functor that computes the residual on the given point x,y between source data and 2D gaussian model.

Computes residual on the given point x,y between the source data and the 2D gaussian model. Required by the ceres solver when performing non-linear least-squares optimisation based on automatic differentiation method.
