Estimate initial 2D gaussian fit to the island using the method of moments.

The moments information used to estimate the 2D gaussian are passed as input parameters.

Parameters
[in]x_bar(double): First moment - x barycentre
[in]y_bar(double): First moment - y barycentre
[in]xx_bar(double): Second moment - xx
[in]yy_bar(double): Second moment - yy
[in]xy_bar(double): Second moment - xy
[in]rms_est(double): RMS estimation
[in]analysis_n_sigma(double): Analysis threshold as multiple of RMS

Fit 2D gaussian to the island.

Fit 2D gaussian to the island using non-linear least-squares optimisation methods implemented by ceres library. Requires image and label map information.

Parameters
[in]data(arma::Mat<real_t>): Image data matrix.
[in]label_map(arma::Mat<int>): Label map matrix.
[in]ceres_diffmethod(CeresDiffMethod): Differentiation method used by ceres library for gaussian fitting.
[in]ceres_solvertype(CeresSolverType): Solver type used by ceres library for gaussian fitting.
