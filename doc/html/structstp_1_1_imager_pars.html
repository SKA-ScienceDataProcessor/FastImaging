Parameters
[in]_image_size(int): Width of the image in pixels. Assumes (image_size/2, image_size/2) corresponds to the origin in UV-space. Must be multiple of 4.
[in]_cell_size(double): Angular-width of a synthesized pixel in the image to be created (arcsecond).
[in]_kernel_function(KernelFunction): Callable object that returns a convolution kernel.
[in]_kernel_support(int): Defines the 'radius' of the bounding box within which convolution takes place (also known as half-support). Box width in pixels = 2*support + 1. The central pixel is the one nearest to the UV co-ordinates.
[in]_kernel_exact(bool): Calculate exact kernel-values for every UV-sample.
[in]_oversampling(int): Controls kernel-generation if 'kernel_exact == False'. Larger values give a finer-sampled set of pre-cached kernels.
[in]_generate_beam(bool): Enables generation of gridded sampling matrix.
[in]_gridding_correction(bool): Corrects the gridding effect of the anti-aliasing kernel on the dirty image and beam model.
[in]_analytic_gcf(bool): Compute approximation of image-domain kernel from analytic expression.
[in]_r_fft(FFTRoutine): Selects FFT routine to be used.
[in]_fft_wisdom_filename(string): FFTW wisdom filename for FFT execution.
