Parameters
[in]_num_timesteps(int): Number of time steps used for A-projection. Set zero to disable A-projection.
[in]_obs_dec(double): Declination of observation pointing centre (in degrees)
[in]_obs_lat(double): Latitude of observation pointing centre (in degrees)
[in]_lha(arma::mat): Local hour angle of visibilities. LHA=0 is transit, LHA=-6h is rising, LHA=+6h is setting.
[in]_mueller_term(arma::mat): Mueller matrix term (defined each image coordinate) used for A-projection.
